    # 什么是闭包

    闭包是一个有权访问另一个函数作用域中变量的函数1。实现方式：函数内部定义其他函数

    function makeAdder(a){
    return function(b){
        return a + b;
    } 
}
var add5 = makeAdder(5);
add5(6); // 11

    产生闭包的根本原因是：作用域对象有被add5引用，因而不会被垃圾回收
